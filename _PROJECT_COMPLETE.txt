## 🎓 HOW THE APPLICATION WORKS

```
1. USER INPUT
   Grade: 4
   Topic: "Types of angles"
   
   ↓

2. GENERATOR AGENT
   Creates:
   • Clear explanation (for grade level)
   • 3 multiple-choice questions
   • 4 options per question
   
   ↓

3. REVIEWER AGENT
   Checks:
   • Age-appropriateness
   • Conceptual correctness
   • Clarity
   
   Status: PASS or FAIL
   
   ↓

4. OPTIONAL REFINEMENT
   If FAIL:
   • Generator gets feedback
   • Re-generates improved content
   
   If PASS:
   • Content is ready
   
   ↓

5. DISPLAY RESULTS
   Shows:
   • Original content
   • Review status
   • Feedback (if any)
   • Refined content (if applicable)
```
